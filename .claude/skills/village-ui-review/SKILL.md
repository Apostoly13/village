Review The Village UI against these principles:

- warm
- calm
- family-safe
- modern
- polished
- simple
- mobile-first
- accessible
- refined, not childish
- welcoming, not cluttered

The Village should feel like a calm parenting support platform, not a generic social media feed.

Design rules:

- Preserve what is already working.
- Improve hierarchy before adding features.
- Reduce clutter.
- Use clear spacing.
- Use readable typography.
- Make CTAs obvious but gentle.
- Make empty states useful.
- Keep Village+ visible but not pushy.
- Respect free vs premium boundaries.
- Avoid unnecessary animations.
- Avoid loud colours or childish styling.
- Day mode should feel warm and welcoming.
- Dark mode should feel soft, like a night light.
- Chat Rooms should feel welcoming, safe, and easy to understand.
- The Village Stall should feel like a trusted community stall, not a commercial marketplace first.

When reviewing a screen:

1. Identify the screen or component.
2. Explain what is working.
3. Explain what is confusing or cluttered.
4. Recommend specific improvements.
5. Check mobile responsiveness.
6. Check day and dark mode.
7. Check accessibility basics.
8. Check whether the screen supports trust and safety.
9. Check whether free vs Village+ boundaries are clear.
10. Avoid major redesigns unless requested.

When making UI changes:

1. Keep changes targeted.
2. Avoid changing unrelated areas.
3. Preserve existing brand direction.
4. Explain before/after impact.
5. Note any follow-up design work.
